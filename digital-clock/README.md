# Digital Clock Application
# Multi-timezone Real-time Clock Display

This directory contains a standalone digital clock application.

## Quick Start

```bash
cd digital-clock
npm install
npm run dev
```

Visit http://localhost:3000 to view the app.

## Features

- Display time in multiple timezones simultaneously
- Add/remove timezones on demand
- Real-time updates every second
- Automatic local storage persistence
- Responsive design for all devices
- Search functionality for timezones
- Dark theme with modern design

## Structure

- `src/app/` - Next.js app and pages
- `src/components/` - React components
- `src/styles/` - Global CSS
- `public/` - Static assets

## Build & Deploy

```bash
npm run build
npm start
```

For production deployment, consider using Vercel, which has native Next.js support.
